# Ukázkový projekt (web)

## Podmínka splnění této lekce
- ✅ Web `cokolada` se zobrazuje na zadané URL na serveru TuX.
- ❌ Pokud ne, oprav.

## Stažení a spuštění
1. Klikni na zelené tlačítko **Code**. Na konci je **Download ZIP**. 
2. Rozbal stažený ZIP → vznikne složka `cokolada-master`.
3. Přejmenuj složku  `cokolada-master` na  `cokolada`.

## O webu cokolada
Web tvoří **několik samostatných souborů** (stránek) a obrázků. Toto je běžná praxe, jinak to ani nejde. Mnoho návštěvníků webu o tom nemá ani tušení. 
Technicky není rozdíl mezi soubory `.html`, `.jpg`, `.png` a dalšími.

Ve správci souborů poklepej na soubor `index.html`. Otevře se ve výchozím prohlížeči (Edge, Chrome). Poznámka: Název "`index`" je vyhrazený pro **výchozí webovou stránku**. 
Projdi si web pomocí tlačítek navigace. Sleduj jak vypadá URL. Na konci bude název zobrazeného souboru.

⚠️ Pro začátek si budeme pamatovat, že kód webové stránky vypadá jinak, než výsledný vzhled – prohlížeč kód (HTML a CSS) převede do vizuální podoby. Obrázky nepřevádí, jen je zobrazí. 

## Statické &times; dynamické stránky 

**Statické stránky** tvoří několik samostatných souborů `.html`. V prohlížeči se mezi nimi přechází pomocí navigace (hypertextových odkazů na všechny ostatní stránky). webový server každému posílá ten samý soubor - každý vidí to samé (HTML a CSS kód je pro všechny stejný).

**Dynamické stránky** webový server sestavuje (generuje) na vyžádání – např. kliknutím na odkaz nebo jinou akcí. Každý může dostat něco jiného. Vyzkoušejte např. na:

- [https://php.edumach.cz/kosik.php](https://php.edumach.cz/kosik.php)
- [https://php.edumach.cz/ohmuv-zakon.php](https://php.edumach.cz/ohmuv-zakon.php)

Pozn: Letos se naučíme tvořit statické, ve 3. ročníku i ty dynamické. V každém případě **ani dynamické stránky se bez HTML a CSS neobejdou**.
